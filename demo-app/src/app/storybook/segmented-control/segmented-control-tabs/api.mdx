## Inputs

### DsSegmentedControl

| Name        | Type               | Default | Description                                                   |
| ----------- | ------------------ | ------- | ------------------------------------------------------------- |
| `tabs`      | `DsSegmentedTab[]` | —       | Array of tab objects to display.                              |
| `activeTab` | `string`           | `''`    | ID of the currently active tab.                               |
| `fullWidth` | `boolean`          | `true`  | Whether the segmented control takes the full container width. |

---

## Outputs

### DsSegmentedControl

| Name        | Type                   | Description                             |
| ----------- | ---------------------- | --------------------------------------- |
| `tabChange` | `EventEmitter<string>` | Emits the tab ID when a tab is clicked. |

---

## Interfaces

### DsSegmentedTab

```typescript
interface DsSegmentedTab {
  id: string; // Unique identifier for the tab
  label: string; // Display text (automatically uppercased in UI)
  count?: number; // Optional count to display in parentheses
}
```

**Example:**

```typescript
const tabs: DsSegmentedTab[] = [
  { id: 'all', label: 'ALL USERS', count: 25 },
  { id: 'active', label: 'ACTIVE', count: 12 },
  { id: 'inactive', label: 'INACTIVE' }, // No count
];
```

## Host Bindings

### DsSegmentedControl

- `class="ds-segmented-control"` — base class applied to the host element

### Internal Elements

- `.ds-segmented-control-container` — wrapper container
- `.ds-segmented-controls` — main control container
  - `class.ds-segment-full-width` — applied when `fullWidth` is true
- `.ds-segment-item` — individual tab button
  - `class.ds-segment-selected` — applied to the active tab
- `.tab-count` — count badge (when count is provided)

---

## Styling

The component uses a retro 2000s aesthetic with:

- **Container**: Light gray background with inset border
- **Tabs**: Outset borders that become inset when active
- **Active State**: White background with blue text
- **Hover State**: Slightly darker background
- **Typography**: Uppercase labels with system fonts
